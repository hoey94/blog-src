1.实现算法

下面定义一个fit方法，实现模型训练。

```python
# 设函数为y=ax+b 求出a,b
def fit(x,y):
  sum_xy = 0
  sum_x = 0
  sum_y = 0
  sum_x2 = 0
  n = x.shape[0]

  for i in range(n):
    sum_xy += x[i]*y[i]
    sum_x += x[i]
    sum_y += y[i]
    sum_x2 += x[i] ** 2

  a = ((sum_xy/n) - (sum_x/n) * (sum_y/n)) / ((sum_x2/n) - (sum_x/n) * (sum_x/n))
  b = (sum_y/n) - (a * (sum_x/n))
  return a,b

```
